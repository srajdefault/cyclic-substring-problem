# Cyclic Substring Problems

**Prepared by:** Sumit Raj

## 📌 Assignment Submission

This repository contains solutions to two algorithmic problems involving cyclic strings and array transformations.

---

## 📂 Problems

### 1. Cyclic Substring Maximum Sum

* Find the maximum sum of a cyclic substring with all unique characters.
* **Approach:** Sliding Window + HashSet + Circular Traversal
  📁 Folder: `Problem1_CyclicSubstringMaxSum`

---

### 2. Array Transformation Cost Minimization

* Make all elements equal using operations of ±K.
* **Approach:** Modulo check + Median-based optimization
  📁 Folder: `Problem2_ArrayTransformationCost`
